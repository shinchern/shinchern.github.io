Flow* Toolbox -
A Platform for the Formal Analysis of Cyber-Physical Systems



Introduction

This site is created to provide an introduction and availability of a toolbox of Flow* which is a formal verification tool for Cyber-Physical Systems. The first version of Flow* was released in 2013 as command line application that accepts a text file including the definition of the CPS model along with the reachability problem on it. The tool has been updated several times to the latest version 2.1.0. The main purpose to introduce a toolbox version of Flow* is to provide a more flexible way for users to
(i) build and analyze complex CPS models, such as the neural network-controlled systems, or
(ii) apply the Flow* reachability algorithms to online platforms such as self-driving vehicles.


Q:   How to use the Flow* toolbox?
A:    The Flow* toolbox provides a set of C++ data structures and functions for modeling the continuous and discrete dynamics in a CPS.
